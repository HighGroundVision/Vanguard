# Vanguard

A Web tool to create Artifact Puzzles

## Prerequisites
 - Node v10.15.0
 - NPM v6.8.0

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
